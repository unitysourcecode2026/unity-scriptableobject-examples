# Unity ScriptableObject Examples

A practical collection of documentation explaining how to use ScriptableObjects in Unity for scalable and maintainable game development.

> This repository contains educational documentation only. No copyrighted Unity Asset Store source code is included.

## Topics

- Player Data
- Enemy Data
- Weapon Data
- Inventory System
- Shop System
- Character Stats
- Save Data
- Best Practices

## Why ScriptableObjects?

ScriptableObjects separate data from game logic, making projects easier to maintain and expand.

### Benefits

- Clean architecture
- Reusable data
- Better organization
- Easier balancing
- Designer-friendly workflow
