# QuestData ScriptableObject

QuestData stores quest information in a reusable ScriptableObject.

## Properties

- Quest Name
- Description
- Objectives
- Rewards
- Required Level
- Completion Status

## Benefits

- Easy quest management
- Reusable quest assets
- Cleaner project structure
- Simple content updates

## Common Use Cases

- RPG Games
- Adventure Games
- Story-driven Games

## Best Practices

- Store quest information only.
- Separate quest logic from quest data.
- Keep rewards configurable.
